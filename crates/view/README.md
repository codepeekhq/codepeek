# codepeek-view

Presentation layer and UI components for codepeek, built on ratatui.

## Scope

- UI components and widgets
- Screen layout and rendering
- Event handling and user interaction
- Depends on `codepeek-core` for domain types

## Not in scope

- Domain logic or shared types -> `codepeek-core`
- Application bootstrap or terminal lifecycle -> `apps/tui`
